# PhishAgent

## Browser extension for real-time phishing detection using machine learning algorithms. Analyzes websites automatically and alerts users to potential threats.

## Features

- Real-time URL and content analysis
- ML-based risk scoring system
- Automatic threat detection
- Instant visual warnings
- Detailed security reports
- Chrome and Firefox support

## Detection Methods

- URL pattern analysis
- Domain reputation checking
- Homograph attack detection
- Login form identification
- Suspicious keyword recognition
- SSL certificate validation
- External link analysis

## Installation

- Download extension files
- Open browser extensions page
- Enable Developer Mode
- Load unpacked extension
- Select extension directory

## Usage
Extension runs automatically on every page visit. Click extension icon to view detailed analysis of current site.


# Türkçe

- Gerçek zamanlı URL ve içerik analizi

- Anında görsel uyarılar

- Chrome ve Firefox desteği

## Tespit Yöntemleri

- URL desen analizi
- Domain itibar kontrolü
- Homograf saldırı tespiti
- Login formu tanımlama
- Şüpheli kelime tanıma
- SSL sertifika doğrulama
- Harici link analizi

## Kurulum

- Eklenti dosyalarını indirin
- Tarayıcı eklentiler sayfasını açın
- Geliştirici Modunu etkinleştirin
